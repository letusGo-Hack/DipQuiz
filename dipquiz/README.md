# Tuist template(CleanArchitecture)
> Tuist version: 3.15.0

## How to run

1. Installing the Tuist
```
bash <(curl -Ls https://install.tuist.io)
```

2. Dependency package fetch
```
tuist fetch
```

3. Open Workspace
```
tuist generate
```

4. Build and run
